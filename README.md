# Introducción
Esto es una aplicación relacionada con los episodios de de la serie de Rick and Morty, en la cual cada usuario pueda gestionar sus episodios vistos y ver estadisticas de estos. 
El objetivo de este proyecto es que usar Firebase Authentication para la gestión de usuarios, así como con Firebase Firestore para el almacenamiento de datos en la nube.
Además, consumir APIs externas con Retrofit, usando la API pública de Rick y Morty para obtener la lista de episodios de la serie y sus datos.
# Características principales
- Autenticación con Firebase
- Listado de episodios con la API de Rick and Morty
- Filtro de episodios vistos y no vistos
- Marcado de episodio como visto
- Ajustes
- Estadísticas
- Peticiones GET con Retrofit
# Tecnologías utilizadas
- Firebase (Auth + Firestore)
- Retrofit
- RecyclerView
- SharedPreferences
- Material Components
- DrawerLayout
# Instrucciones de uso
- Cómo clonar
`git clone https://github.com/jcpenfie/RickandMorty.git`
- Configurar Firebase
1. Te creas una cuenta de firebase
2. Entas en la consola
3. Creas un nuevo proyecto y sige las instrucciones paso a paso
4. Añades el google-service.json en la versión Proyect app>src
5. Una vez creado agregas una app de android
6. Dentro de la app entramos en Compilación > Autenticación > Método de acceso y habilitas mínimo el de correo electrónico
