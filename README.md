# Introducción
Esto es una aplicación relacionada con los episodios de de la serie de Rick and Morty, en la cual cada usuario pueda gestionar sus episodios vistos y ver estadisticas de estos. 
El objetivo de este proyecto es que usar Firebase Authentication para la gestión de usuarios, así como con Firebase Firestore para el almacenamiento de datos en la nube.
Además, consumir APIs externas con Retrofit, usando la API pública de Rick y Morty para obtener la lista de episodios de la serie y sus datos.
# Características principales
autenticación, listado de episodios, filtro, marcado como visto, ajustes, estadísticas, etc.
# Tecnologías utilizadas
Firebase (Auth + Firestore), Retrofit, RecyclerView, SharedPreferences, Material Components
